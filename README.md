# Vendor Performance Analysis

## Project Overview
This project analyzes vendor performance using sales, delivery time, and vendor rating data. The goal is to identify high-performing vendors, detect delivery delays, and help businesses make better supplier decisions.

## Problem Statement
Companies often work with multiple vendors for supplying products. However, not all vendors perform equally. Some vendors deliver products on time and generate high sales, while others may cause delays or have low performance. This project helps analyze vendor data to identify reliable vendors and improve overall supply chain efficiency.

## Dataset
The dataset contains vendor-related information including:

- Vendor ID
- Product Category
- Order Quantity
- Sales Amount
- Delivery Time
- Vendor Rating

This data helps evaluate vendor performance based on sales contribution, delivery efficiency, and customer ratings.

## Tools and Technologies Used
- SQL – Data extraction and querying
- Python – Data cleaning and preprocessing
- Power BI – Data visualization and dashboard creation

## Analysis Performed
The following analysis was performed in this project:

- Total sales generated by each vendor
- Vendor performance comparison
- Delivery delay analysis
- Vendor rating analysis
- Identification of top-performing vendors

## Dashboard Visualization
An interactive Power BI dashboard was created to visualize vendor performance. The dashboard includes:

- Sales by Vendor
- Vendor Rating Comparison
- Delivery Delay Analysis
- Top Vendors by Sales

## Key Insights
- A small group of vendors contributes a large portion of total sales.
- Some vendors have higher delivery delays which may affect supply chain efficiency.
- Vendors with higher ratings tend to have better sales performance and reliability.

## Conclusion
Vendor performance analysis helps businesses identify reliable vendors and monitor vendor efficiency. Using data analysis and visualization tools, companies can make informed decisions to improve vendor management and overall business performance.
